# DNA File Converter Tool

A tool for converting raw DNA data files between major genealogy platforms.

## Supported formats:
- 23andMe
- AncestryDNA
- MyHeritage
- FamilyTreeDNA

## Purpose:
Enable compatibility between different DNA services.
