# MyHeritage DNA Format

MyHeritage uses a CSV-based format for raw DNA data.

## Structure

Fields:

- RSID
- CHROMOSOME
- POSITION
- RESULT

### Example:

rs3094315,1,742429,AA  
rs12562034,1,758311,GG  
rs3934834,1,995669,CT  

---

## Notes

- CSV format (comma-separated)
- Header row is usually present
- Genotype stored in RESULT column

---

## Usage

Used for genealogy analysis and compatibility with other DNA tools.
