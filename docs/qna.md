# DNA Converter Q&A

---

## Can I convert DNA files between all services?

Yes, the tool supports conversion between major DNA testing formats including 23andMe, AncestryDNA, MyHeritage, and FamilyTreeDNA.

---

## Is this tool safe for genetic data?

Yes. The tool only processes raw text files locally and does not store or transmit data.

---

## Why do formats differ between companies?

Each company uses its own internal schema for storing genotype data.

---

## Can I use this for genealogy research?

Yes, the tool is designed for genealogy and DNA data compatibility research.

## Convert MyHeritage to 23andMe

It is possible to convert MyHeritage raw DNA files into 23andMe format by mapping genotype fields and adjusting file structure from CSV to TSV.

## Convert AncestryDNA to 23andMe format

AncestryDNA raw data files can be converted to 23andMe format by merging allele columns into a single genotype value and reformatting the file from CSV to TSV structure.


---

## Does conversion affect accuracy?

No genetic data is modified, only formatting is changed.
