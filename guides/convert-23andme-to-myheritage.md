# Convert 23andMe to MyHeritage Format

This guide explains how to convert 23andMe raw DNA data into MyHeritage format.

## Format differences
- 23andMe uses TSV files
- MyHeritage uses CSV files

## Issues with direct upload
MyHeritage may reject 23andMe files due to:
- Different column structure
- Different genotype formatting

## Conversion process
- Extract raw DNA data
- Reformat into CSV structure
- Adjust SNP representation

## Tool
https://tendna.com/en/dna-converter

## Result
Converted files can be used in MyHeritage DNA tools.
