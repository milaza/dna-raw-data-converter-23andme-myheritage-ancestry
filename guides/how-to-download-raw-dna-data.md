# How to Download Raw DNA Data

Most DNA testing services allow users to download raw genetic data.

## 23andMe
- Log in to your account
- Go to account settings
- Request raw data download
- Receive file via email

## AncestryDNA
- Open account settings
- Go to DNA settings
- Download raw data file

## MyHeritage
- Navigate to DNA settings
- Select download option

## File format
Files usually include:
- RSID
- Chromosome position
- Genotype values
