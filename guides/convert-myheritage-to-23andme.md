# Convert MyHeritage to 23andMe Format

This guide explains how MyHeritage raw DNA data differs from 23andMe format and how conversion works.

## Why conversion is needed
MyHeritage and 23andMe use different file structures:
- MyHeritage: CSV format
- 23andMe: TSV format

Because of this, files are not directly compatible.

## Key differences
- Column order is different
- Genotype representation varies
- SNP coverage may differ

## Conversion process
A typical conversion includes:
- Reformatting columns
- Adjusting genotype encoding
- Ensuring RSID compatibility
