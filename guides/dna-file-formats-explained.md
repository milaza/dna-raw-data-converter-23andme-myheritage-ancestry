# DNA File Formats Explained

Different DNA testing companies use different file formats.

## Common formats

### 23andMe
- TSV format
- Single genotype column

### AncestryDNA
- CSV format
- Separate allele columns

### MyHeritage
- CSV format
- Different SNP structure

## Why formats differ
Each company uses its own pipeline for:
- SNP processing
- genotype encoding
- database structure

## Compatibility issue
Because of differences, raw DNA files are not interchangeable.
